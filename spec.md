# SPEC: Whispers from the Void

## Overview
A haunted anonymous confession app where users communicate through a supernatural Ouija board interface. Confessions appear as spectral whispers from the void, creating an unsettling yet captivating experience.

## Core Features

### 1. Ouija Board Input System
- Interactive planchette follows cursor with ghostly trail
- Letters glow with ethereal light when selected
- Spectral sound effects on each letter selection
- Automatic message composition with possession-style animation
- "GOODBYE" to submit confession to the void

### 2. Confession Feed (The Void)
- Confessions materialize from darkness like spirits
- Each confession appears in decaying, handwritten fonts
- Fade in/out with ectoplasm effects
- Random positioning creates chaotic, haunted atmosphere
- Older confessions decay and vanish into the void

### 3. Atmospheric Horror Elements
- Flickering candlelight illumination
- Floating spectral particles
- Ambient whispers and creaking sounds
- Screen glitches and distortions
- Shadow figures moving in periphery
- Blood moon background
- Fog/mist effects

### 4. Technical Features
- Real-time confession updates (SSE)
- Anonymous submission (no tracking)
- Profanity filter (keep it spooky, not offensive)
- Rate limiting (prevent spam)
- Responsive design (haunted on all devices)

## Design System

### Color Palette
- Primary: #0a0a0a (Void Black)
- Secondary: #1a1a2e (Midnight Blue)
- Accent: #8b0000 (Blood Red)
- Glow: #00ff41 (Spectral Green)
- Text: #e0e0e0 (Ghost White)

### Typography
- Headers: "Creepster", cursive
- Body: "Special Elite", monospace (typewriter)
- Confessions: "Shadows Into Light", cursive (handwritten)

### Animations
- Flicker: Random opacity changes (candles)
- Float: Vertical drift (spirits)
- Decay: Gradual fade + blur (old confessions)
- Glitch: RGB split + distortion
- Possession: Erratic typing effect
